# RTL-to-GDS Implementation of 8-bit ALU

## Overview

Implemented RTL-to-GDSII flow for an 8-bit ALU using OpenLane and Sky130 PDK.

## Tools Used

* OpenLane
* Sky130 PDK
* KLayout
* Magic VLSI

## Results

* No DRC violations
* No LVS errors
* Successful GDS generation

## Files

* results → final layout (GDS, DEF, etc.)
* reports → synthesis & timing reports
* images → layout screenshots
